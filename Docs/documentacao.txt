                        Baixa o packt json

yarn init -y (Baixa o packt json)

yarn add -D  typescript nodemon ts-node @types/express @types/node (baixa o nodemon e o type)

yarn add express pg typeorm dotenv reflect-metadata (baixando postgresql, orm do type)


					Inicializando o TypeScript

npx tsc --init

                     Inicializando a aplicação

yarn dev                   

                        Gerando as migration

yarn migration:generate


                        Criando a migration

yarn migration:run

yarn add express-async-errors
